# Zheng Shang

I work on reproducible research workflows for veterinary parasitology,
Toxoplasma literature review, and Manubot-style scientific writing automation.

## Current Focus

- Manubot-style manuscript workflows for transparent research writing.
- Evidence tables and citation metadata checks for literature reviews.
- Reproducible manuscript validation with CI-friendly commands.
- Public/private boundary checks for research repositories.
- Documentation that helps researchers reproduce, review, and maintain scholarly
  writing workflows.

## Public Work

- [`manubot-paper`](https://github.com/SHzzzAyys/manubot-paper): a public

  Toxoplasma research.
- [`oss-contribution-log`](https://github.com/SHzzzAyys/oss-contribution-log):
  a public log of open-source contribution targets, issues, pull requests, and

## Contribution Direction

I am focusing external contributions on the Manubot and open-science ecosystem,
especially:

- manuscript build and validation documentation;
- generated tables and evidence-table workflows;
- PubMed, PMC, DOI, and citation warning diagnostics;
- reproducible research repository hygiene;
- privacy boundaries for public research materials.

## Working Principles

- Prefer accurate, inspectable claims over adoption or impact claims without
  public evidence.
- Include reproducible commands, environment details, and verification notes in
  issues and pull requests.
- Keep private academic drafts, local workspace paths, and private contact
  information outside public repositories.
- Keep pull requests small enough for maintainers to review quickly.
